Students Grades web service API

  • Get all students and grades:

    • in order to get all the student list and their grades use getAllStudent() function.

    ⇨ Function signature:

    https://students-grades.herokuapp.com/getAllStudents

  • Get specific student grades:

    • in order to get the grades of specific student use getStudGradeById(id) function.

    ⇨ Function signature:

    https://students-grades.herokuapp.com/getStudGradeById/:userid

    ⇨ Example:

    https://students-grades.herokuapp.com/getStudGradeById/301427647

  • Get list of students from a particular year:

    • in order to get list of students from a particular year use getStudentsByYear(year) function.

    Please notice that the argument "year" must be between 1 to 4.

    1- is first year students, 2 is second year students, et cetera...

    ⇨ Function signature:

    https://students-grades.herokuapp.com/getStudentsByYear/:year

    ⇨ Example:

    https://students-grades.herokuapp.com/getStudentsByYear/3

** All of these functions returns a JSON object.
